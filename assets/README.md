# AVD Demo Assets

This directory holds screenshots, diagrams, and recordings used as fallback materials during live demos.

## Directory Structure

```
assets/
├── diagrams/         # Architecture and flow diagrams
├── screenshots/      # Portal screenshots for each demo
│   ├── demo1-*       # Image Build Pipeline
│   ├── demo2-*       # Host Pool Deployment
│   ├── demo3-*       # FSLogix Configuration
│   ├── demo4-*       # GPU Verification
│   └── demo5-*       # Monitoring Setup
└── recordings/       # Short video clips (30–60 sec each)
```

## Screenshot Naming Convention

Use: `demo{N}-{description}.png`

Examples:
- `demo1-gallery-versions.png`
- `demo1-image-template.png`
- `demo2-session-hosts.png`
- `demo2-health-checks.png`
- `demo3-registry-keys.png`
- `demo3-profile-share.png`
- `demo4-gpu-partition.png`
- `demo4-nvidia-smi.png`
- `demo5-avd-insights.png`
- `demo5-kql-results.png`
- `demo5-cost-analysis.png`

## Capture Instructions

1. Set browser zoom to 125–150%
2. Use full-window screenshots (not snippets)
3. Capture at 1920×1080 minimum resolution
4. Remove any sensitive data (subscription IDs, usernames) or blur them
5. Save as PNG
